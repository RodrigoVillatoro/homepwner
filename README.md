# homepwner
## An app that keeps records of items. 

Homepwner is an app that keeps an inventory of all your possessions. The objective of this app is to build a realistically complex application which includes the use of UITableViews, UINavigationController, Camera, Saving and Loading Data, etc. 

![homepwner preview](https://github.com/RodrigoVillatoro/homepwner/blob/master/homepwner.png)

This is an exercise from the book [iOS Programming](https://www.bignerdranch.com/we-write/ios-programming/), by The Big Nerd Ranch.
